# LifeOS — Personal Life Dashboard PWA

A high-performance Progressive Web App built with React, Tailwind CSS, and Lucide Icons. Glassmorphism dark mode design.

## Features

- **Dashboard** — Sleep score, steps, focus hours, savings rate with animated charts
- **Profile** — Personal stats, achievements, Q1 focus areas
- **Settings** — iOS-style toggles for notifications, reminders, privacy
- **PWA** — Installable, offline-ready with Stale-While-Revalidate caching
- **Mobile-first** — Responsive from 320px to 1440px

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Lucide Icons
- Service Worker (offline support)
- GitHub Pages (auto-deploy via Actions)

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

Push to `main` branch — GitHub Actions automatically builds and deploys to GitHub Pages.
